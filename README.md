# Proyecto Final - Sistema de Gestión de Productos

Programa de consola en Python para gestionar un catálogo de productos, usando SQLite como base de datos.

## Funcionalidades
- Agregar productos (nombre, categoría, precio)
- Mostrar todos los productos
- Buscar productos por nombre
- Eliminar productos por ID

## Cómo ejecutarlo
1. Cloná el repositorio
2. Corré `python main.py`

No requiere instalar librerías externas (sqlite3 viene incluido con Python).
